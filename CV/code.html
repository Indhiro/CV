Box #2 Deskripsi Diri

Tentang saya

Saya sekarang sedang menduduki semester 4 di Universitas Udayana dengan prodi Teknologi Informasi. Teknologi Informasi adalah istilah umum untuk teknologi apa pun yang membantu manusia dalam membuat, mengubah, menyimpan, mengomunikasikan dan/atau menyebarkan informasi. Teknologi Informasi pasti akan banyak dibutuhkan di masa depan jadi, menurut saya prodi ini sangat baik untuk ditekuni.

Untuk hobi saya sendiri, saya senang menghabiskan waktu luang saya dengan bermain game dan menonton film. Dengan melakukan hal tersebut, waktu terasa lebih cepat dan juga lebih menyenangkan. Bermain game bersama teman dapat membantu saya melepaskan penat dan stress setelah melakukan perkuliahan.

Mengingat perkuliahan offline sudah dimulai, saat ini saya berada di rumah sewa bersama teman saya. Disini saya dipaksa untuk keluar dari zona nyaman karena semua harus dilakukan dengan mandiri. Mulai dari makan, bersih-bersih , mencuci pakaian, mengatur keuangan, dan lainnya.
